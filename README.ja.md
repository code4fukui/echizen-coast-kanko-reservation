# 越前海岸エリア 宿泊予約状況

福井県越前海岸エリアの宿泊予約状況を分析するためのオープンデータプロジェクトです。福井県観光連盟の「FTAS」システムのデータを活用し、オンラインで確認できるようになっています。

## 主な機能
- 越前海岸エリアのホテル数や総客室数の表示
- 宿泊予約データの可視化（客室稼働率、平均宿泊単価など）
- 日付範囲の選択による分析
- CSVデータのダウンロード

## データ・API
- 福井県観光連盟の「FTAS」システムからのオープンデータ
- `latest_hotel.csv`: ホテルの客室数データ
- `latest_rsv_sum.csv`: 宿泊予約の集計データ
